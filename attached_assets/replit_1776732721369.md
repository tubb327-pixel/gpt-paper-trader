**Use this as the design + build brief you hand to Replit Agent.** Generate a working Next.js 14 app that designs itself and ships. The goal is a single live-updating dashboard monitoring a Solana pump.fun paper-trading bot on GCP. Nothing exists yet — build from scratch.

## Data source

Everything comes from a single backend API I already run. You do not touch Postgres.

**Upstream base URL:** `https://34.186.65.190.sslip.io`

**Auth:** every upstream request needs `Authorization: Bearer <TOKEN>`. **Do NOT put the token in client code.** Store it in Replit Secrets as `DASHBOARD_API_TOKEN`. The frontend calls your Next.js route handler at `/api/proxy/*`; the route handler attaches the token and forwards to upstream.

### Server-side proxy — `app/api/proxy/[...path]/route.ts`

```ts
import { NextRequest, NextResponse } from 'next/server';

const UPSTREAM = 'https://34.186.65.190.sslip.io';

export async function GET(
  req: NextRequest,
  { params }: { params: { path: string[] } },
) {
  const token = process.env.DASHBOARD_API_TOKEN;
  if (!token) {
    return NextResponse.json({ error: 'DASHBOARD_API_TOKEN unset' }, { status: 500 });
  }
  const path = params.path.join('/');
  const url = `${UPSTREAM}/${path}${req.nextUrl.search}`;
  const res = await fetch(url, {
    headers: { Authorization: `Bearer ${token}` },
    cache: 'no-store',
  });
  const body = await res.text();
  return new NextResponse(body, {
    status: res.status,
    headers: { 'Content-Type': res.headers.get('content-type') ?? 'application/json' },
  });
}
```

### Typed fetchers — `lib/api.ts`

```ts
export async function getJSON<T>(path: string): Promise<T> {
  const res = await fetch(`/api/proxy${path}`, { cache: 'no-store' });
  if (!res.ok) throw new Error(`${path} ${res.status}`);
  return res.json();
}

export const api = {
  wallet:         () => getJSON<Wallet>('/api/wallet'),
  positions:      () => getJSON<Position[]>('/api/positions'),
  closedTrades:   (limit = 50) => getJSON<ClosedTrade[]>(`/api/trades/closed?limit=${limit}`),
  exitReasons:    () => getJSON<ExitReasonRow[]>('/api/exit-reasons'),
  devs:           () => getJSON<DevRow[]>('/api/devs'),
  unknowns:       (limit = 25) => getJSON<DevRow[]>(`/api/unknowns?limit=${limit}`),
  recentLaunches: (minutes = 30) => getJSON<LaunchRow[]>(`/api/launches/recent?minutes=${minutes}`),
  systemHealth:   () => getJSON<SystemHealth>('/api/system/health'),
};
```

### Value conventions

- Fields ending in `_lam` are lamports. Divide by `1e9` for SOL.
- Prices (`entry_price`, `current_price`, `exit_price`) are **lamports per atomic token unit**. Pump.fun tokens have 6 decimals, total supply 1B whole.
- **USD market cap = `price × 1e6 × sol_usd`**. Pull `sol_usd` from `/api/system/health`.
- Format SOL to 3 decimal places (`1.234 SOL`). Format USD with magnitude suffixes (`$4.2k`, `$1.2M`, `$3.1B`).
- Timestamps ISO 8601 UTC. Display in US Central (UTC-6) — don't use the browser locale.

## Polling

Use TanStack Query with a 5000ms refetch interval for each data source. In `app/layout.tsx` wrap the tree in a `QueryClientProvider`.

```ts
const queryClient = new QueryClient({
  defaultOptions: {
    queries: { refetchInterval: 5000, refetchOnWindowFocus: true, staleTime: 0 },
  },
});
```

In `dashboard/client.tsx` fire 8 queries in parallel:

```ts
const wallet   = useQuery({ queryKey: ['wallet'],   queryFn: api.wallet });
const positions= useQuery({ queryKey: ['positions'],queryFn: api.positions });
// ...etc
```

Age / clock on open positions ticks every second independently of the 5s query cycle — use `useEffect` + `setInterval(1000)` and render age from `entry_at`.

On network failure: keep last-known values, don't wipe the screen. Flash a slim banner at the top of the wallet card, keep polling.
